1)Kiedy powstał JavaScript?(Podaj date)
    4 grudnia 1995 rok


    2)Dlaczego jest to język skryptowy?
    Ponieważ jego kod źródłowy jest wykonywany linia po linii w czasie rzeczywistym, a nie kompilowany przed uruchomieniem.

    3)Gdzie jest wykonywany JS? (klient lub serwer)
    Jest wykonywany po stronie klienta, głównie w przeglądarkach internetowych użytkowników.

    4)Dlaczego jest JS to bezpieczne narzędzie?
    JS jest bezpieczne jeżeli, przeglądarki stosują zabezpieczenia, takie jak sandboxing, które pomagają chronić użytkowników przed potencjalnymi zagrożeniami

    5)Czy jest to język obiektowy , jeśli tak to dlaczego.
    JavaScript jest językiem obiektowym. W JavaScript wszystko jest obiektem. Programy JavaScript są oparte na manipulowaniu obiektami, tworzeniu nowych obiektów i korzystaniu z metod i właściwości istniejących obiektów.

    6)Wymień zalety JS:
    Zaleta1 Łatwość nauki i dostępność zasobów edukacyjnych
    Zaleta2 Możliwość tworzenia dynamicznych i interaktywnych stron internetowych
    Zaleta3 Wysoka popularność


    7)Framework to:
    Narzędzie lub struktura, która zapewnia gotowe rozwiązania i narzędzia do tworzenia aplikacji. W przypadku JavaScript, frameworki są często używane do ułatwienia tworzenia aplikacji internetowych.

    8)Do aplikacji mobilnych można użyć Framework
    Tak, do tworzenia aplikacji mobilnych można używać frameworków JavaScript, takich jak React Native czy NativeScript.

    9)Aplikacje responsywne to:
    aplikacje internetowe lub mobilne, które automatycznie dostosowują się do różnych rozmiarów ekranów i urządzeń, aby zapewnić użytkownikom optymalne doświadczenie.

    10)Przepisz linie kodu pod nimi wytłumaczenie jak działa ta ten fragment kodu<
    input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

    Ten fragment kodu HTML definiuje przycisk o nazwie "przycisk", który wyświetla napis "Nowa Strona" i nie ma zdefiniowanej akcji JavaScript do wykonania po kliknięciu przycisku. Dlatego, w jego aktualnej postaci, nie będzie on wykonywał żadnych widocznych działań po kliknięciu.

    11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

    toolbar=no - ukrywa przyciski katalogów

    directories=no - oznacza, że przyciski nawigacyjne nie będą widoczne

    menubar=no - oznacza, że pasek menu nie będzie widoczny

    height=280 - wysokość okna będzie wynosić 280 pikseli

    width=160 - szerokość okna będzie wynosić 160 pikseli

    top=200 - Ten atrybut określa pozycję okna w pionie w pikselach

    left=200 - lewa krawędź okna będzie oddalona o 200 pikseli od lewej krawędzi ekranu

    12)Opisz następującą instrukcję window.close()

    Instrukcja window.close() jest używana w języku JavaScript do zamknięcia bieżącego okna przeglądarki internetowej.
